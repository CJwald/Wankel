\section{Wankel}

Wankel is a game engine written to support games with procedural mesh generation in mind. It contains a SandboxApp example showcasing a simple application.

\section{Features}

Wankel supports the following:

\begin{itemize}
    \item Mouse and Keyboard Inputs
    \item Controller Inputs
    \item Event System
    \item ImGui
    \item Entt ECS 
    \item Linux only so far
    \item Camera system
    \item Renderer (OpenGL) with simple mesh rendering
    \item Simple physics system
    \item AABB collisions

\end{itemize}
